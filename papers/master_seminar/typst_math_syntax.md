# Typst 数式・文法ガイド

出典: [Typstで数式を記述しよう【Typstシリーズ その4】](https://oucrc.net/articles/ii7iq5ffruz/)

---

## 1. 数式モードの開始 / 終了

| 用途 | 記法 | 例 |
|------|------|----|
| インライン数式 | `$ ... $` | `一次関数は $y = a x + b$ と表される。` |
| ディスプレイ数式 | `$ ... $`（前後に半角スペース） | `$ y = a x + b $` |

> **ポイント:** 単一文字はそのまま表示。複数文字は変数・関数扱いになるため、`a x` のようにスペースを入れるか、`"const"` のように文字列にする。

---

## 2. 改行・整列

```typst
$ sum_(k=1)^(n) k &= 1 + 2 + 3 + ... + n \
  &= 1/2 n(n + 1) $
```
- `\`：数式内改行  
- `&`：列のそろえ位置（複数列揃え）

---

## 3. 添え字・上付き

| 機能 | 記法 | 例 |
|------|------|----|
| 上付き | `^` | `$ a^2 $` |
| 下付き | `_` | `$ x_1 $` |
| 複数文字 / 複合添え字 | `()` | `$ x_(k-1) $` |

---

## 3.1 アクセントの詳細

ハットはhat(x), バーはoverline(x),微分のドットはdot(x)
---

## 4. よく使う関数・演算子

| 種類 | 記法 | 例 |
|------|------|----|
| 平方根 / n 乗根 | `sqrt(x)` / `root(n, x)` | `$ sqrt(3) $`, `$ root(3, a) $` |
| 絶対値 / ノルム | `abs(x)` / `norm(x)` | `$ abs(-3) $`, `$ norm(vec(1,2)) $` |
| 三角関数 | `sin x`, `cos x`, `tan x` | `$ sin pi = 0 $` |
| 極限 | `lim_(x->0)` | `$ lim_(x->0) sin x / x = 1 $` |
| 総和 | `sum_(i=1)^(n)` | `$ sum_(i=1)^(n) i $` |
| 積分 | `integral_(a)^(b)` | `$ integral_(0)^(1) x^2 d x $` |
| 集合記号 | `in`, `in.not`, `subset`, ... | `$ x in {1,2,3} $` |

---

## 5. 分数と括弧

```typst
$ frac(1, 2) + frac(1, 3) = frac(5, 6) $
$ (a + b)/(c + d) $
```
- `frac(a, b)` : 分子・分母を縦に表示  
- `/`           : 簡易表記

---

## 6. ベクトル・行列

| 機能 | 記法 | 例 |
|------|------|----|
| ベクトル | `vec(a, b, c)` | `$ vec(1,2,3) $` |
| 行列   | `mat( a11, a12; a21, a22 )` | `$ mat(1, 2; 3, 4) $` |
| ドット積 | `.dot` または `·` | `$ vec(a) dot vec(b) $` |

---

## 7. コード例（複数行整列）

```typst
$ sum_(k=1)^(n) k &= 1 + 2 + ... + n \
                  &= (n(n + 1))/2 $
```

---

## 8. よくある落とし穴

1. **空白不足** — `ax` は未定義関数扱い。`a x` と書くか `"ax"` と文字列にする。
2. **ディスプレイ数式のスペース** — `$` と式の間にスペースを忘れない：`$ y = ... $`。
3. **無限大記号** — `oo`（小文字 o を 2 つ）で `∞` を表す。
